# SpotifySongPlayer
A device utilizing an ESP32, TFT Display, and the Spotify API to display, pause, and switch songs that you are currently listening to. It also displays the current album cover.

### Actual hardware:
* Lolin C3 Mini (ESP32)
* 1.8" ST7735 (TFT Display)
* Wires
### Libraries utilized:
* [Spotify API Wrapper for ESP32](https://github.com/FinianLandes/SpotifyEsp32/blob/main/src/SpotifyEsp32.cpp#L963)
* [TFT_eSPI library](https://github.com/Bodmer/TFT_eSPI)
* [TJpeg_Decoder](https://github.com/Bodmer/TJpg_Decoder)
